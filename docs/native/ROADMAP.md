# Juno Native — Execution Roadmap

1. Audit and handoff baseline — in progress.
2. Shared API/auth/sync/storage/search packages and contract drift gate.
3. Independent macOS shell, chat, search, and account flows.
4. macOS Juno Code, trust/permissions, Remote Host, and controlled Computer Use.
5. Independent adaptive iOS/iPadOS shell, chat, sidebar, search, and account flows.
6. Mobile Juno Code Cloud and Remote, diffs, approvals, events, and push lifecycle.
7. Cross-surface convergence, offline/conflict/E2E, performance, accessibility, and localization.
8. Native CI, Release archives, signing/notarization/TestFlight, GitHub release, and production download metadata.

Each phase must end with focused tests, a reviewed diff, a coherent commit, and updated `STATUS.md`, `HANDOFF.md`, and `handoff.json` before the next phase begins.
